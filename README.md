# Weather App (Expo + React Native)

A location-centric weekly weather app built with Expo's default React template style.

## Features
- Uses your current location to fetch weather forecast.
- Weekly forecast list with clickable day cards.
- Detailed weather view for selected day.
- Blue + dark blue theme for a clean, modern UI.
- Powered by the Open-Meteo API.

## Run locally
```bash
npm install
npm run start
```

Then open with Expo Go on mobile, or run `npm run web` for browser preview.
